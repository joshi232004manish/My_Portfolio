# Portfolio Website - Manish Joshi

## 🚀 Overview

This is my personal **portfolio website** showcasing my skills, projects, and experience as a **developer**. The website is built using **HTML, CSS, JavaScript, and React.js**, with a focus on clean UI, responsiveness, and smooth animations.

## 🛠️ Technologies Used

- **Frontend:** React.js, HTML5, CSS3, JavaScript
- **Backend (if applicable):** Node.js, Express.js
- **Database (if applicable):** MongoDB
- **Hosting & Deployment:** Vercel / Netlify / GitHub Pages
- **Other Integrations:** Cloudinary (for image handling), EmailJS (for contact form)

## 🎨 Features

✔️ **Responsive Design** – Works seamlessly on desktop & mobile devices. ✔️ **Interactive UI** – Smooth animations & transitions for a better user experience. ✔️ **Project Showcase** – A dedicated section displaying my best projects with links. ✔️ **Skills & Experience** – Highlights my technical skills and professional journey. ✔️ **Contact Form** – Users can reach out via a functional email form. ✔️ **Social Media Links** – Direct links to LinkedIn, GitHub, and other platforms.

## 🔧 Installation & Setup

1. **Clone the repository**

```sh
 git clone https://github.com/joshi232004manish/portfolio-website.git
```

2. **Navigate to the project folder**

```sh
 cd portfolio-website
```

3. **Install dependencies**

```sh
 npm install
```

4. **Run the development server**

```sh
 npm start
```

The site will be available at `http://localhost:3000/`.
